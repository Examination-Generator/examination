## Tips for Best UX

### Visual Hierarchy
1. Subject name (largest)
2. Paper names (medium)
3. Topics/sections (smaller)
4. Buttons (icon-based, colored)

### Spacing
- Papers: 1rem gap
- Topics/Sections: 0.5rem gap
- Modal padding: 1.5rem
- Button spacing: 0.75rem

### Feedback
- Hover: Background color change
- Focus: Ring outline
- Click: Button press effect
- Save: Alert message
- Error: Alert message

### Accessibility
- All inputs labeled
- Buttons have titles
- Keyboard navigable
- Screen reader friendly
- High contrast mode compatible
